## 🚀 **INSTALLATION & SETUP**

### **1. Prerequisites**
```bash
# Python 3.8+ required
python3 --version

# Install system dependencies
sudo apt update
sudo apt install python3-pip python3-venv
```

### **2. Clone & Setup**
```bash
# Create project directory
mkdir ULTRA_QUANTUM_AGENT
cd ULTRA_QUANTUM_AGENT

# Copy all files to directory
# ultra_config_v3.json
# ULTRA_ADVANCED_AGENT_v3.py
# ultra_requirements_v3.txt
```

### **3. Install Dependencies**
```bash
# Create virtual environment
python3 -m venv quantum_env
source quantum_env/bin/activate

# Install core dependencies
pip install psutil requests colorama numpy

# Install advanced dependencies (optional)
pip install -r ultra_requirements_v3.txt
```

### **4. Configuration**
```bash
# Edit ultra_config_v3.json with your Perplexity AI key
# The file is already configured with your API key
```

### **5. Run the Quantum Agent**
```bash
# Activate virtual environment
source quantum_env/bin/activate

# Run the quantum agent
python3 ULTRA_ADVANCED_AGENT_v3.py
```
